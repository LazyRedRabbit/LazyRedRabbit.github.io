Java 기본 설정

2018-02-03
Java

기본경로를 설정해주는 이유는 어떤 경로에서라도 실행이 가능하게 하기 위해서이다

자바를 설치하고 난 후 기본적으로 경로 설정을 해주어야 이클립스 같은 개발툴에서 바로 컴파일이 가능하다

기본 설정을 해줌으로써 개발툴에서 자바의 기능을 사용할 수 있다

// 아래 링크에서 필요한 버전의 자바를 다운로드 한다
http://www.oracle.com/technetwork/java/javase/archive-139210.html

공식 홈페이지 자바 버전별 다운로드
www.oracle.com
나중에는 JDK 버전별로 별도로 모아놓는 것이 편리함

ex) D:javajdk[버전], D:developjdk[버전]

// 기본경로로 설정

내 컴퓨터 -> 고급 시스템 설정 -> 고급 -> 환경변수

변수 이름 : JAVA_HOME

변수 값 : [자바 설치 경로]bin

  • 윈도우 10에서는 아래처럼 해주면 된다

// path 값을 찾아 수정

변수 값 : ;%JAVA_HOME%bin; 추가 ( 기존 값 삭제하지 말고 우측 끝에 추가할 것 !! )

  • 윈도우10에서는 변수 (PATH)를 더블클릭하면 아래처럼 편집기가 나온다

// CLASSPATH 설정

Javac로 자바소스를 컴파일하면 .class 파일 생성되는데

이것을 구분하고 불러내기 위하여 Class 환경 변수도 설정해 주어야 한다.

새로 만들기

변수 이름 : CLASSPATH

변수 값 : .;%JAVA_HOME%\lib\tools.jar

여기서 유의해야 할 것은 앞에 .; 이다.

. 은 현재 디텍토리에서 파일을 찾으라는 얘기이고

; 은 두 경로를 구분 짓는 것이다.

// 세팅 후 확인

시작 -> 실행 -> cmd

커맨드 창에서 java -version 입력 후 엔터

커맨드 창에서 echo %CLASSPATH% 입력 후 엔터

커맨드 창에서 javac 입력 후 엔터

이렇게 기본 설정이 끝났다
